## 0x04. 摸鱼守则

为了保持长期的低压运行，严格遵守以下守则：

1. 不追求全情投入，只在间隙进行
2. 难题超过 10 分钟没思路，直接看答案或者放弃，拒绝焦虑
3. 过程 > 结果，哪怕只写了一行 `print()`，也是胜利
